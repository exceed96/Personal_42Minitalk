# 📜42_minitalk - 개인 프로젝트


##  ☑️ "minitalk"과제란?
<div align="center">
  signal함수를 이용하여 프로세스간의 UNIX신호 데이터 주고받는 프로그램을 만드는 프로젝트
</div>
